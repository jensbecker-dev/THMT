## Installation

1. Clone repository

```
git clone https://github.com/jensbecker-dev/THMT
```

2. Install depenecides

```
cd THMT/backend
pip install -r requirements.txt
python3 create_db.py
```

3. Start the application

```
python3 app.py
```
